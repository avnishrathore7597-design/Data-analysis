# Sales Data Analysis Using Pandas and Matplotlib

## Project Overview

This project demonstrates basic data analysis and visualization using Python, Pandas, Matplotlib, and Seaborn. The dataset contains monthly sales and profit information. The project performs statistical analysis and generates visualizations to gain business insights.

## Features

* Load CSV dataset
* Display dataset information
* Calculate average sales and profit
* Generate Bar Chart
* Generate Scatter Plot
* Generate Heatmap
* Provide business insights

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn

## Installation

pip install -r requirements.txt

## Run

python data_analysis.py

## Visualizations

* Sales Bar Chart
* Sales vs Profit Scatter Plot
* Correlation Heatmap
